So verwendest du die App

  1. Lege die App-Dateien an einen beliebigen Ort.
  2. Lege daneben deinen Ordner images mit Unterordnern an.
  3. Klicke auf images-Ordner auswählen und wähle genau diesen Ordner.
  4. Klicke einen Unterordner an.

Verhalten pro Unterordner:

  • Kein Bild: Alert mit Fehlermeldung
  • Bild, aber keine cover.json: Editor zum Erstellen neuer Cover-Zonen
  • Bild und cover.json: Viewer mit grauen Zonen zum Aufdecken/Abdecken
